# Future Features

Deferred capabilities that are intentionally out of scope for the current build, captured
here so the rationale and intended shape aren't lost. Each feature lives in its own file;
add a one-line entry below and link it when you record a new one.

These are **not** committed work — they are a backlog of "we deliberately chose not to do
this yet, and here's why / how we'd do it later." When one is picked up, it graduates into
an OpenSpec change (`openspec/changes/`) and its entry here is updated to point at that
change (or removed once archived).
